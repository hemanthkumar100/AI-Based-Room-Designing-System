# AI-Based Room Design System

## Description
AI-Based Room Design System is a full-stack web application that uses AI-based image processing to detect furniture from room images and generate automated interior design layouts using user preferences or predefined rules.

## Technologies Used
- Frontend: Angular, HTML, CSS, TypeScript  
- Backend: Node.js, Express.js  
- AI: TensorFlow.js (YOLO / COCO-SSD)  
- Database: MongoDB  

## Workflow
1. Upload room image  
2. Detect objects using AI  
3. Provide preferences or use auto-design  
4. Generate room design layouts  
5. Save selected design  

## Advantages
- Automated interior design  
- Saves time and cost  
- Personalized layouts  
- Scalable system  

## Future Scope
- 3D visualization  
- AR/VR integration  
- Advanced generative AI
